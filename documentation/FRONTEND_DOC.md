# Стек фронтенд-разработки: Treasury System

Проект базируется на современном индустриальном стеке, ориентированном на создание высоконадежных финтех-интерфейсов и работу со сложными структурами данных.

## 1. Фундамент (Core)
* **React 19:** Основная библиотека для построения компонентного интерфейса. Позволяет создавать быстрые SPA (Single Page Applications) с реактивным обновлением данных.
* **Vite:** Сверхбыстрый сборщик нового поколения. Обеспечивает мгновенный запуск сервера разработки и эффективную сборку проекта.
* **TypeScript:** Строгая типизация. Минимизирует риск ошибок при работе с API, гарантируя соответствие типов данных (например, сумм платежей, дат и статусов) между бэкендом и фронтендом.

## 2. Дизайн и UI-Kit (Interface)
* **Ant Design (AntD):** Профессиональная библиотека UI-компонентов.
    * **Преимущества:** Банковский стандарт визуализации данных. 
    * **Основные модули:** Мощные таблицы с фильтрацией/сортировкой, формы с автоматической валидацией, модальные окна и индикаторы процессов.
    * **Стиль:** Строгий корпоративный интерфейс, оптимизированный под плотные потоки информации.

## 3. Работа с данными и API (Network & State)
* **Axios:** HTTP-клиент для взаимодействия с FastAPI. Реализован через централизованный `apiClient` с механизмом интерцепторов (автоматическая подстановка JWT-токенов).
* **TanStack Query (React Query):** Управление серверным состоянием. 
    * Автоматическое кэширование данных.
    * Фоновая синхронизация реестра платежей с базой данных.
    * Управление состояниями загрузки (Loading) и обработки ошибок.

## 4. Навигация (Routing)
* **React Router:** Система маршрутизации. Управляет переходами между страницами (Авторизация, Реестр, Календарь) без перезагрузки браузера, обеспечивая бесшовный пользовательский опыт.
